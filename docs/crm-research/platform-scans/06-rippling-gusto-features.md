# Rippling & Gusto Features Research

## RIPPLING

### 1. Payroll Processing

#### Core Features
- Automated data flow from HR
- Global multi-currency (185+ countries, 50+ currencies)
- Automatic tax calculation and filing
- Multi-entity management
- Real-time payroll previews

#### Integration
- Automated journal entries (NetSuite, SAP)
- Role-based access controls
- Multi-level approval workflows
- Immutable audit logs

### 2. HR Workflows

#### Automation Capabilities
- Configurable onboarding workflows
- Document automation with e-signatures
- Workflow Studio for custom triggers
- 100+ out-of-the-box automations
- 50%+ reduction in HR admin time

### 3. Compliance-Grade Approvals

#### Features
- Multi-level approval chains
- Conditional routing logic
- Dynamic re-routing (vacation coverage)
- Complete audit logging
- Role-based approval paths

### 4. Employee Lifecycle

#### Coverage
- Onboarding (benefits, documents, devices)
- Active employment (salary, benefits, performance)
- Qualifying Life Events (auto-detection, auto-updates)
- Offboarding (device revocation, final paycheck)

#### Self-Service Portal
- View personal info and pay stubs
- Update contributions and benefits
- Manage life events

### 5. Benefits Administration

#### Plan Coverage
- Medical, Dental, Vision, 401K
- Commuter, FSA, HSA
- 4,000+ plans from leading carriers

#### Automation
- New hire auto-enrollment
- Eligible plan filtering
- QLE auto-detection
- COBRA, ACA, ERISA compliance

### 6. Time Tracking

#### Compliance Features
- Auto-applies federal, state, local laws
- Break and rest policy enforcement
- Overtime threshold enforcement
- New state auto-policy creation

#### Attendance
- Mobile timeclock app
- Biometric devices (fingerprint, badge, facial)
- Manager approval workflows
- Compliance 360 dashboards

### 7. Platform Architecture

#### Unified Data Model
- Single Employee Graph
- Seamless data flow
- No manual data entry
- 600+ app integrations

#### Pricing
- $21-$29 per employee per month
- Implementation: 5-15% of annual fees

## GUSTO

### 1. Payroll Automation

#### Core Features
- Unlimited payroll runs
- AutoPilot processing
- Multiple payment methods (direct deposit, debit cards, checks)
- Multi-state support (all 50 states)
- Mobile payroll management

### 2. Tax Filing

#### Federal Forms
- Form 944/941, Form 940, FUTA
- Form 8974 (tax credits)
- W-2s and 1099s auto-generated

#### Features
- Auto-calculation and payment
- State tax registration (all 50)
- Tax reconciliation adjustments
- ATF for employees via April partnership
- **Free with all plans**

### 3. Benefits Administration

#### Coverage
- Medical, dental, vision
- HSA, FSA, 401(k)
- Commuter, life, disability
- 3,500+ benefit options

#### Automation
- No manual entry
- Auto-deduction calculation
- Expert enrollment guidance
- Continuous support

#### Compliance
- COBRA, HIPAA, ACA reporting
- Section 125 filing
- ERISA compliance

### 4. HR Compliance

#### Compliance Hub
- Centralized dashboard
- Proactive alerts on law changes
- In-app guidance

#### Coverage Areas
- FLSA, ACA, OSHA, FMLA
- Paid leave policies
- State-specific training

#### Resources
- SHRM/HRCI-certified experts
- Hundreds of policy templates
- Handbook templates

#### Security
- HIPAA compliant
- SOC 2 Type II certified

### 5. Employee Self-Service

#### Payroll
- Access pay stubs and W-2s
- Multiple direct deposit accounts
- Lifetime tax document access
- Gusto Debit Card

#### Time & Attendance
- Clock in/out
- Submit timesheets
- Request time off
- View PTO balances

#### Onboarding
- Pre-day-one account creation
- Secure document upload
- E-sign forms (W-4, I-9)
- Org charts and directories

### 6. Approval Workflows

#### Payroll Approvals
- Multi-admin support
- Email notifications
- Approve, decline, or edit
- AutoPilot auto-disable when enabled

#### Custom Roles
- Role delegation
- Limited access grants
- Simple: 1 role
- Plus: 5 roles
- Premium: Unlimited
