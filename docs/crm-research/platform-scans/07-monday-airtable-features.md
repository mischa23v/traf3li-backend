# Monday.com & Airtable Features Research

## MONDAY.COM

### 1. Visual Work Views (27+)

#### Kanban View
- Drag-and-drop cards
- Workflow stage columns
- Real-time collaboration
- Agile and marketing use cases

#### Timeline View
- Horizontal bar visualization
- Duration and workload tracking
- Milestone support
- Long-term planning

#### Gantt View (Pro+)
- Task dependencies with arrows
- Critical path tracking
- Milestone markers (diamonds)
- Subitems support

#### Calendar View
- Daily, weekly, monthly displays
- Deadline tracking
- Schedule management

#### Additional Views
- List, Chart, Map, Workload
- Total 27+ view types

### 2. Formula Columns (Pro+)

#### Capabilities
- No-code calculations
- AI Formula Assistant
- Compatible with 20+ column types

#### Features
- IF function (most common)
- Mathematical and logical operations
- Cannot directly trigger automations (use third-party apps)

### 3. Automations

#### Standard Plan
- 250 actions/month
- Pre-built automation recipes
- Real-time processing via mondayDB

#### Use Cases
- Status updates
- Notifications (Slack, email)
- Item creation
- Deadline actions
- Data sync

### 4. Dashboards

#### Features
- Up to 30 widgets per dashboard
- Run reports from connected boards
- Real-time data display
- Google Data Studio integration

#### Widgets
- Standard visualizations
- Custom widget development
- Third-party embeds
- Text widgets (unlimited)

### 5. Integrations

#### Native
- Slack, Google Drive, Microsoft 365
- Jira, Canva, Calendly
- HubSpot, Tableau, Geckoboard

#### Developer Tools
- REST and GraphQL APIs
- SDKs for multiple languages
- Custom app framework
- Private apps

### 6. mondayDB

#### Architecture
- Built for scalability
- Real-time data availability
- Extended API capabilities
- Supports large-scale boards

## AIRTABLE

### 1. Views (6+)

#### Grid View (Default)
- Spreadsheet-like layout
- Sorting, filtering, grouping

#### Kanban View
- Cards by single select field
- Drag-and-drop management

#### Calendar View
- Date field visualization
- Event planning

#### Gallery View
- Card-based with images
- Portfolio display

#### Gantt View (Pro+)
- Project timelines
- Duration visualization

#### Timeline View (Pro+)
- Linear date-based view

### 2. Formula Fields

#### Capabilities
- Numbers, dates, strings, booleans
- AI-powered formula generation
- Named references (not cell coordinates)

#### Key Limitations
- Cannot reference other records
- Workaround: Self-linking + lookup

#### Functions
- Mathematical, text, date
- Array functions (with rollups)
- Regular expressions (RE2)

### 3. Linked Records

#### Relationship Types
- One-to-One
- One-to-Many
- Many-to-Many

#### Features
- Reciprocal relationships
- Data consistency
- Self-linked records
- Lookup fields for cross-table data

### 4. Automations

#### Traditional
- Task automation
- Google, Slack, social integrations
- JavaScript extensions

#### AI-Powered (2025)
- Omni conversational AI builder
- AI automation actions
- Airtable Agents for orchestration

### 5. Interfaces

#### Interface Designer
- No-code app builder
- Record lists, charts, galleries
- Form viewers, custom buttons
- Conditional display

#### AI Tools
- Cobuilder: Natural language app creation
- Omni: Conversational AI builder
- Production-ready apps in seconds
